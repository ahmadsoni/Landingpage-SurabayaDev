- ✨ TypeScript
- 💨 Tailwind CSS 3

- 🤖 Meta tag
## Cara Menjalankan

### 1. Clone landing page ini menggunakan cara berikut:
You can start the server using this command:

```bash
git clone https://github.com/ahmadsoni/Landingpage-SurabayaDev.git
```
### 2. NPM install:
```bash
npm install
```
### 3. Run program :
```bash
npm run dev
```
### 3. Buka Link Local :
👉 http://localhost:1234/ 👈
